# Cube Quest

Cube Quest is a responsive frontend web project built using HTML and CSS. This project was created to practice web development fundamentals, responsive design, and modern UI styling techniques.

## 🚀 Features

- Responsive design
- Modern user interface
- Clean and structured layout
- Mobile-friendly design
- Deployed using GitHub Pages

## 🛠️ Technologies Used

- HTML5
- CSS3
- Git
- GitHub Pages

Through this project, I practiced:

- HTML page structure and semantic elements
- CSS styling and layout techniques
- Responsive web design
- Version control using Git and GitHub
- Website deployment using GitHub Pages
